## 프로젝트 규칙
- **커밋·푸시 전 민감정보 스캔 필수** (전역 CLAUDE.md 루틴): 특히 텔레그램 봇 토큰·chat_id·개인 이메일
- 수집 실패는 치명적(수집 자체가 목적) — 스크래퍼·필터 변경 시 실제 실행 결과로 검증하고, 공고 건수가 기존 대비 급감하면 원인 규명 전 배포 금지
- 필터 변경 시 재무·회계·세무 외 직무(마케팅 등)가 새어 들어오는지 표본 확인
- 테스트: `PYTHONPATH=. python -m unittest discover -s tests`

    (assert가 없어 회귀 게이트 역할도 못 한다). CI는 `--ignore`로 제외하므로, 로컬에서
    빠르게 돌리려면 `python -m pytest tests --ignore=tests/test_wanted_debug.py`를 쓸 것
